# whatareyoudoingtoday

### 개발환경
IDE: Android studio
### 사용한 프론트 기술
1. 프레임워크: flutter
2. 언어:Dart
   
### 사용한 파이어베이스 기술
1. firebase auth
   - 사용자의 계정에 대해서 보다 쉽게 다룰 수 있게 해줌
     - 로그인 및 회원가입
     - pw 변경
2. firebase firestore
   - 여러 할 일이나 사용자에 대한 정보를 저장함
